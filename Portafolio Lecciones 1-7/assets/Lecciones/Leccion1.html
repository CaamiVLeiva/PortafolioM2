Lección 1

1. ¿Qué se entiende por desarrollo web?.

El desarrollo web es crear sitios web o aplicaciones para internet, desde diseñarlos hasta mantenerlos, asegurandose de que funcionen correctamente en distintos dispositivos y navegadores.

2. Diferencias entre Front-End, Back-End y Fullstack.

Front End Back End Full Stack
Diseño Visual del sitio web Procesamiento de datos en el servidor Desarrollo tanto de Front-End como del Back-End
Creación y organización del contenido Gestion de base de datos Capacidad para trabajar en todas las capas de desarrollo de una aplicación o página web
Implementación de funcionalidades que se ejecutan en el navegador. Implementación lógica detras de escena de la aplicación o página web Conocimiento integral de la arquitectura y funcionamiento completo de una aplicación o página web.

4. El rol del Navegador.

Interpreta el código HTML de una página web y lo convierte en la interfaz visual que vemos. Algunas funciones cruciales son:

  1. Interpretación: Lee el código HTML y comprende cómo estructurar y mostrar el contenido.
  2. Renderizado: Utiliza el CSS para aplicar estilos y presenta la página web de manera visualmente atractiva.
  3. Detección de Errores: Conoce y muestra los errores de programación de una página.

5. ¿Qué es la W3C?.

La World Wide Web Consortium (W3C) es una organización sin fines de lucro que se encarga de definir las normas y protocolos para los desarrolladores para garantizar una web universalmente accesible, más allá de las diferencias de idioma, navegador, sistema operativo o plataforma.

6. Evolución del HTML hacia el HTML5.

Tres características importantes introducidas en HTML5:

  • Mejoras en formularios: Introduccion de nuevos tipos de entrada y validaciones.
  • Semántica Mejorada: Agregó elementos como '<header>','<footer>','<nav>', entre otros.
  • Soporte multimedia: Permitió la inclusión de audio y video nativo en las páginas web.

7. La triada HTML, CSS y Javascript: contenido, presentación y comportamiento.

Ejemplos de como utilizar HTML, CSS Y Javascript para gestionar el contenido, la presentación y el comportamiento en una página web:

  1. Contenido (HTML): Define la estructura y el contenido de la página.
  2. Presentación (CSS): Estiliza el contenido para hacerlo visualmente atractivo.
  3. Comportamiento (JS): Agrega interactividad y funcionalidad dinámica a la página.

8. Utilizar el potencial de un editor de texto para el desarrollo.

Características clave de VSCode que facilitan el desarrollo web:

  • Resaltado de sintaxis: Ayuda a identificar y entender el código con colores distintivos.
  • Autocompletado: Proporciona sugerencias de código y completado automático para aumentar la productividad.
  • Extensiones: Permite personalizar el editor con funciones adicionales para el desarrollo web.

9. Conociendo el inspector de elementos en un navegador.

  • Abrir el Inspector: Click derecho en cualquier parte de la página y selección "Inspeccionar"
  • Utilizar el Inspector: Navega por el código HTML para examinar elementos, estilos y contenido de la página web de manera interactiva.
  • Modificar Elementos: Selecciona algun elemento en HTML o CSS para modificar los elementos.
